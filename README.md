# Disease Prediction App 🏥

This project is a **Machine Learning based Disease Prediction Web App** built using **Python, Streamlit, and Logistic Regression**.

## Features
- Predicts disease based on health parameters
- Simple user interface using Streamlit
- Machine Learning model using Logistic Regression
- Interactive inputs for prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Input Features
- Age
- Gender
- Chest Pain (cp)
- Resting Blood Pressure (trestbps)
- Cholesterol (chol)
- Maximum Heart Rate (thalach)
- Exercise Induced Angina (exang)
- Oldpeak

## How to Run

1️⃣ Install requirements
2️⃣ Run the app

## Output

The model predicts whether a patient is likely to have a disease based on input parameters.
